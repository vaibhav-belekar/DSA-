## ⚙️ How to Run

1. Clone the repository

   ```bash
   git clone https://github.com/vaibhav-belekar/DSA-.git
   cd DSA-
   ```

2. Compile a program (example: `linear_search.cpp`)

   ```bash
   g++ 2_ARRAY/linear_search.cpp -o linear_search
   ./linear_search
   ```
